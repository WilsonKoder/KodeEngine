KodeEngine Development State.

KodeEngine is in very, very early stages of development, with no implementation of a Physics or Audio API yet. The rendering API is still being actively developed, with ease of use a key goal in the development of our API's. KodeEngine will eventually be a full game engine, but for now it is closer to a wrapper of OpenGL functions.
